# cz-format-extension

🚧 WIP 🚧

## install

```
yarn add cz-format-extension commitizen -D
```

## Usage

パスを通すために、`.czrc` もしくは `package.json`に記述します

### .czrc

```
{
  "path": "cz-format-extension"
}
```

### package.json
```
"config": {
  "commitizen": {
    "path": "cz-format-extension"
  }
}
```
